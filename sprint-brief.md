## How the APK gets built — the full pipeline

This is the chain from your React code to a running app on a phone. Every step must succeed before the next one works.

```
React + Vite (your code)
        │
        │  npm run build
        ▼
   dist/  (static web files: HTML + JS + CSS)
        │
        │  npx cap sync android
        ▼
   android/  (native Android project — auto-generated by Capacitor)
   android/app/src/main/assets/public/  ← your dist/ copied here
        │
        │  Android Studio → Build → Build APK
        │  (or: npx cap run android  to build + launch in one step)
        ▼
   app-debug.apk  (installable on emulator or physical device)
        │
        │  adb install  /  Android Studio "Run" button
        ▼
   App running on Hackcessible_Pixel6 emulator (or real phone)
```

**Rule:** every time you change React code, you must run **both** of these before the APK reflects the change:
```bash
npm run build          # rebuilds dist/
npx cap sync android   # copies dist/ into the Android project
```
Then in Android Studio: click the green **Run ▶** button (or `Shift+F10`).  
You do NOT need to rebuild the APK from scratch each time — Android Studio does incremental builds.

### One-time Android Studio setup (do this before the sprint, not during it)
```bash
# 1. Confirm Android Studio is installed and SDK is present
adb --version                          # should print a version number
echo $ANDROID_HOME                     # should print a path like ~/Android/Sdk

# 2. Install Capacitor packages (run from inside jaki/)
npm install @capacitor/core @capacitor/cli @capacitor/android

# 3. Initialise Capacitor (run once — skips if capacitor.config.ts already exists)
npx cap init "Jaki" "com.hackcessible.jaki" --web-dir dist

# 4. Add the Android platform (run once — creates the android/ folder)
npx cap add android

# 5. Build the web app and sync into Android
npm run build
npx cap sync android

# 6. Open Android Studio
npx cap open android
```

Inside Android Studio:
- Wait for Gradle sync to finish (bottom status bar)
- Tools → Device Manager → Create Device → Pixel 6 / API 33 (if not already done)
- Click the green **Run ▶** button → select `Hackcessible_Pixel6` → app launches

### Every sprint iteration after that
```bash
# From inside jaki/ — run this after any code change
npm run build && npx cap sync android
# Then hit Run ▶ in Android Studio
```

### To get a shareable APK file (e.g. to send to a mentor's phone)
In Android Studio: **Build → Build Bundle(s) / APK(s) → Build APK(s)**  
Output goes to: `android/app/build/outputs/apk/debug/app-debug.apk`  
Send this file to a physical Android device and tap it to install (device must have "Install from unknown sources" enabled in Settings).

- **Files to touch:** `src/screens/Login.tsx` (new), `src/App.tsx`, `src/types.ts`, `src/lib/supabase.ts`

**1C. Capacitor Android build — get it on the emulator**
